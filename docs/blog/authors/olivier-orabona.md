  <div class="author-bio">
    <h2>About</h2>
    <p>Cloud architect, cybersecurity consultant, and software engineer with over 20 years of hands-on experience building production systems. Olivier founded O2 Consulting Services &amp; Investment (O2CSI), a Paris-based technology consultancy specializing in cloud infrastructure, security auditing, and custom software development. His clients span finance, healthcare, and SaaS companies across Europe.</p>
    <p>Olivier created astix after years of frustration watching AI coding assistants waste thousands of tokens reading raw source files instead of understanding code structure. The result: a semantic code intelligence platform that parses 38 languages into an AST-backed call graph, reducing token consumption by 128x. astix is now open source under Apache 2.0 and powers code intelligence for teams using Claude Code, Cursor, and other MCP-compatible assistants.</p>
    <p>He started programming at age 9 on an Amstrad CPC464 with a tape recorder. Some things never change — he still prefers terminals over GUIs.</p>
